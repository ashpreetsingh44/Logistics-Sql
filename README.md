# Logistics-Sql

A Logistics Management System built using **Oracle SQL** and **PL/SQL** featuring a normalized relational database design with stored procedures and triggers to automate order processing, inventory management, payment handling, shipment tracking, and delivery operations.

## Features

- Normalized relational database schema
- Stored procedures for order and delivery management
- Triggers for driver assignment, payment status, and inventory updates
- Shipment tracking and delivery status management
- Sample data for testing and verification

## Tech Stack

- Oracle SQL
- PL/SQL
- Oracle Database

## Project Structure

```text
01_tables.sql
02_procedures.sql
03_triggers.sql
04_sample_data.sql
```

## Run

Execute the SQL files in the following order:

1. `01_tables.sql`
2. `02_procedures.sql`
3. `03_triggers.sql`
4. `04_sample_data.sql`
